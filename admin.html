신청자 관리

날짜별 정원 설정

파티 날짜 관리

  • 날짜 추가: 날짜를 선택하고 표시명(선택)을 입력한 뒤 '날짜 추가'를 누릅니다.
  • 추가된 날짜는 신청 페이지의 날짜 선택란에 자동으로 표시됩니다.
  • 허용 지점: 날짜별로 오픈할 지점만 선택할 수 있습니다. 아무 지점도 체크하지 않으면 전체 지점이 열립니다.
  • 각 날짜 옆 수정/삭제 버튼으로 관리할 수 있습니다.

정원 & 상태 설정

  • 각 날짜(요일)별로 정원(명)을 설정합니다.
  • 상태 드롭다운: 자동 / 모집중 / 마감임박 / 마감 중 선택
  • 자동: 신청자 수 ÷ 정원 비율에 따라 상태가 자동 계산됩니다.
  • 수동 설정: 직접 선택한 상태가 자동 계산보다 우선 적용됩니다.
  • 정원을 0명으로 설정하면 자동으로 마감 처리됩니다.

임계값 & 배지

  • 마감임박 기준: 신청 비율이 이 값 이상이면 '마감임박' (기본 80%)
  • 마감 기준: 이 값 이상이면 '마감' (기본 100%)
  • 잔여석 배지 문구: 메인 페이지 상단에 표시되는 배지 텍스트 (비워두면 자동 생성)
상태 변경 안내
• 수동 설정: 상태 드롭다운에서 직접 변경한 값이 최우선 적용됩니다.
• 자동 계산: 수동 설정이 없으면, 신청자 수 ÷ 정원 비율에 따라 자동으로 결정됩니다.

파티 날짜 관리

신청 페이지에 표시되는 날짜를 관리합니다. 기본 금/토/일 외에 공휴일 등 추가 파티 날짜를 설정할 수 있습니다.

지점/가격 관리

지점 관리

  • 지점 추가: 하단 '＋ 지점 추가' 버튼을 눌러 새 지점을 만듭니다.
  • 각 지점 카드에서 이름 확인, 삭제 버튼으로 제거할 수 있습니다.

가격 설정

  • 남성/여성 참가비: 지점별 1부 참가비를 원 단위로 입력합니다.
  • 비고: '포틀럭 포함' 등 가격 관련 메모를 추가할 수 있습니다.
  • 2부 기본가: 2부 참여 시 추가 금액 (원)
  • 2부 선결제 할인율: 1부+2부를 함께 결제할 때 적용되는 할인 (%)

가격 미리보기

  • 입력값을 변경하면 하단 미리보기에서 모든 조합의 최종 금액을 실시간 확인할 수 있습니다.
  • 변경 후 반드시 '지점/가격 저장' 버튼을 눌러야 반영됩니다.

지점별 참가비, 2부 옵션을 설정합니다. 변경 후 반드시 '저장' 버튼을 눌러주세요.

할인코드 관리

할인코드 만들기

  • 할인코드: 직접 입력하거나 '자동생성' 버튼으로 랜덤 코드를 만듭니다.
  • 할인 종류: 금액 할인(원) 또는 비율 할인(%) 중 선택합니다.
  • 할인값: 금액 할인이면 원 단위(예: 5000), 비율이면 %(예: 10)를 입력합니다.
  • 사용 제한: 최대 사용 가능 횟수 (0 = 무제한).

관리

  • 발급된 코드 목록에서 사용 현황(사용 횟수/제한)을 확인할 수 있습니다.
  • 활성/비활성 토글로 코드를 일시 중지하거나 다시 활성화합니다.
  • 삭제 버튼으로 코드를 영구 삭제합니다.
  • 참가자가 신청 시 할인코드를 입력하면 자동으로 할인이 적용됩니다.

할인코드를 만들어 참가자에게 공유하세요. 신청 시 할인코드를 입력하면 자동으로 할인이 적용됩니다.
